# Dell-inspiron-13-7370-Hackintosh

# 如果你要使用这个EFI，你需要在BIOS中关闭 Secure Boot，否则无法启动

## 配置
CPU | i5-8250U
------------ | -------------
主板 | 戴尔06KDFD
内存 | 8GB（2400MHz）
主硬盘 | 西数SN750
显卡 | UHD620
声卡 | ALC295
网卡 | AX200

## 正常驱动
1.核显工作正常

2.声卡工作正常

3.Type-C似乎工作正常

4.快捷键工作正常

5.网卡工作正常

6.触摸版工作正常

7.背光键盘工作正常

8.睡眠正常

## 无法工作的
1.蓝牙未被识别到

2.耳机口无法识别

3.变频似乎不是很正常，忽高忽低

4.读卡器未驱动
